# Tic Tac Toe Game

## Project Description

A simple interactive Tic Tac Toe game built with React.js.

This project was created as a learning project to practice React components, state management, event handling, and game logic.

The game allows two players to play alternately using X and O symbols on a 3x3 board. The system automatically detects the winner and provides move history navigation.

This project helped improve my understanding of frontend logic and React application structure.

## Main Features

* Interactive 3x3 game board
* Player turns between X and O
* Winner detection
* Move history tracking
* Jump back to previous moves
* Dynamic game state updates

## Technologies Used

* React.js
* JavaScript
* SCSS / CSS

## Learning Focus

* React class components
* State management
* Event handling
* Conditional rendering
* Basic game logic
* Component structure
